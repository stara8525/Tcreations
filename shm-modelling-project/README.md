# Modelling Simple Harmonic Motion with Python

## Project Overview

This project will explore simple harmonic motion using Python simulations and visualizations.

The goal is to model how parameters such as mass, spring constant, damping, and driving frequency affect the motion of an oscillator.

## Planned Topics

- Basic simple harmonic motion
- Mass-spring systems
- Damped oscillations
- Driven oscillations and resonance

## Tools

- Python
- NumPy
- Matplotlib
- Jupyter Notebook
