## Projects

### Breast Cancer Diagnosis Analysis

Explored whether physical measurements of tumor cell nuclei can help distinguish between benign and malignant tumors using Python, pandas, matplotlib, and scikit-learn.

Key skills:
- Data exploration
- Feature comparison
- Logistic regression
- ROC AUC model evaluation
- Biomedical data interpretation

[View project](./breast-cancer-diagnosis-project)

### Detector Signal Analysis and Monte Carlo Simulation

Simulated detector-like signals and analyzed how noise, detection thresholds, signal amplitude, and timing-window requirements affect event detection.

This project was inspired by detector-analysis concepts used in particle, nuclear, and rare-event physics experiments.

Key skills:
- Signal simulation
- Gaussian noise modelling
- Threshold-based event detection
- Monte Carlo simulation
- Detection efficiency analysis
- False positive analysis

[View project](./detector-signal-analysis)

---

### Modelling Simple Harmonic Motion with Python

Modelled simple harmonic motion and mass-spring oscillators using Python simulations and visualizations. The project explores angular frequency, period, velocity, acceleration, energy conservation, damping, and resonance.

Key skills:
- Physics modelling
- NumPy simulations
- Matplotlib visualizations
- Parameter analysis
- Energy conservation
- Damped and driven oscillations

[View project](./shm-modelling-project)
